# דשבורד מנהלים (CRM פנימי) — מצב עבודה

> ענף `feature/admin-dashboard` · עודכן 2026-06-17

## מה נבנה ✅

### Backend (NestJS) — תוספות
רוב ה-API ל-CRM כבר היה קיים. הושלמו הפערים:
- **מודול `dashboard`** — `GET /api/dashboard/summary` (staff/admin): תור טיפול,
  משרות פעילות + מספר מועמדים שהוצגו, תזכורות פתוחות, וסטטיסטיקות (קו"ח השבוע,
  גיוסים החודש, עמלות פתוחות בש"ח, תזכורות באיחור, מנויי מייל פעילים).
- **רשומות שיחה** — `POST /api/candidates/:id/calls` (הוספת CallLog ידנית).
- **פרטי מועמד/משרה מורחבים** — `findOne` כולל כעת שיחות, הצגות (עם שם המשרה/המועמד)
  וגיוסים.
- **מודול `mailing`** — `GET /api/mailing/subscribers` (סינון תחום/אזור/סטטוס),
  `POST /api/mailing/send` (שליחה לרשימה מסוננת, **חסום בשבת/יו"ט**).

### Frontend (Next.js 14, RTL מלא) — חדש
- תשתית: פרוקסי `app/api/admin/[...path]` (same-origin), `lib/admin-api.ts`,
  מיפוי תוויות עברית `lib/labels.ts`, מראה מעברי סטטוס `lib/status-machine.ts`,
  עזרי עמלה `lib/commission.ts`. רכיבי UI: `Select`, `Textarea`, `StatusBadge`,
  `StatCard`, `Feedback` (Loading/Empty/Error/Success/PageHeader).
- **שער כניסה** — `app/admin/layout.tsx`: רק `staff`/`admin`, אחרת ניתוב החוצה.
  קישור "דשבורד צוות" בהדר מוצג רק לצוות.
- **לוח בקרה** (`/admin`) — כל הווידג'טים מסעיף 7.1.
- **מועמדים** (`/admin/candidates`, `/[id]`) — רשימה+חיפוש+סינון; כרטיס מלא: פרטים,
  שינוי סטטוס (מעברים חוקיים בלבד), הערות פנימיות, הורדת קו"ח (signed URL),
  הסכמת מייל, היסטוריית שיחות + הוספה, משרות שהוצג אליהן.
- **משרות** (`/admin/jobs`, `/new`, `/[id]`) — רשימה; יצירה עם הפרדת תיאור
  פנימי/ציבורי + טיוטה־או־פרסום; עריכה, מעברי סטטוס, מועמדים שהוצגו + הצגת מועמד.
- **מעסיקים** (`/admin/employers`) — רשימה + הוספה (פנימי בלבד).
- **עמלות** (`/admin/commissions`, `/[id]/invoice`) — טבלה עם סטטוס גיוס+עמלה,
  סימון "ניתן לגבות" אחרי ערבות, מעבר `pending→invoiced→paid`, תזכורת גבייה,
  ומסך חשבונית להדפסה.
- **תזכורות** (`/admin/reminders`) — רשימה, יצירה, סימון כטופל.
- **רשימת תפוצה** (`/admin/mailing`) — מנויים + סינון, שליחה ידנית (חסומה בשבת),
  ייצוא CSV.
